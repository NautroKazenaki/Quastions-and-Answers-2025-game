# Медиа-файлы для новогодней викторины

Эта папка содержит все медиа-файлы (изображения и видео), используемые в вопросах и ответах викторины.

## Структура папок

```
media/
├── questions/       # Медиа для вопросов
│   ├── round1/      # Изображения/видео для раунда 1 (100-500 очков)
│   ├── round2/      # Изображения/видео для раунда 2 (200-1000 очков)
│   └── supergame/   # Медиа для супер игры
└── answers/         # Медиа для ответов
    ├── round1/      # Изображения для ответов раунда 1
    ├── round2/      # Изображения для ответов раунда 2
    └── supergame/   # Медиа для ответов супер игры
```

## Поддерживаемые форматы

### Изображения
- `.jpg` / `.jpeg`
- `.png`
- `.gif`
- `.webp`

### Видео
- `.mp4`
- `.webm`

## Как добавить медиа-файлы

### 1. Подготовка файлов

Поместите ваши изображения или видео в соответствующую папку:
- Для вопросов первого раунда → `questions/round1/`
- Для ответов второго раунда → `answers/round2/`
- И так далее

### 2. Обновление questions.json

Откройте файл `src/data/questions.json` и добавьте ссылки на медиа-файлы:

#### Пример 1: Простой вопрос с текстовым ответом

```json
{
  "text": "Ваш вопрос здесь",
  "answer": "Правильный ответ здесь"
}
```

#### Пример 2: Вопрос с изображением и текстовым ответом

```json
{
  "text": "Ваш вопрос здесь",
  "media": {
    "type": "image",
    "src": "/media/questions/round1/your-image.jpg"
  },
  "answer": "Правильный ответ здесь"
}
```

#### Пример 3: Вопрос с видео и текстовым ответом

```json
{
  "text": "Ваш вопрос здесь",
  "media": {
    "type": "video",
    "src": "/media/questions/round1/your-video.mp4"
  },
  "answer": "Правильный ответ здесь"
}
```

#### Пример 4: Вопрос с ответом, содержащим картинку

```json
{
  "text": "Ваш вопрос здесь",
  "answer": {
    "text": "Правильный ответ здесь",
    "media": {
      "type": "image",
      "src": "/media/answers/round1/answer-image.jpg"
    }
  }
}
```

#### Пример 5: Вопрос с картинкой и ответ с картинкой

```json
{
  "text": "Ваш вопрос здесь",
  "media": {
    "type": "image",
    "src": "/media/questions/round1/question-image.jpg"
  },
  "answer": {
    "text": "Правильный ответ здесь",
    "media": {
      "type": "image",
      "src": "/media/answers/round1/answer-image.jpg"
    }
  }
}
```

### 3. Важные замечания

- **Все вопросы должны иметь поле `answer`** — это обязательно!
- **Пути всегда начинаются с `/media/`** — это обязательно!
- Путь относительно папки `public`, т.е. файл `public/media/questions/round1/image.jpg` в JSON указывается как `/media/questions/round1/image.jpg`
- Поле `answer` может быть:
  - **Строкой** для простых текстовых ответов: `"answer": "Текст ответа"`
  - **Объектом** для ответов с медиа: `"answer": { "text": "...", "media": {...} }`
- Поле `media` опционально — вопросы могут быть чисто текстовыми
- Размер файлов: старайтесь оптимизировать медиа для веба
  - Изображения: рекомендуется до 500KB
  - Видео: рекомендуется до 5MB

## Примеры использования

### В основных раундах

```json
{
  "name": "Новогодние традиции",
  "round": 1,
  "questions": [
    {
      "text": "Простой текстовый вопрос",
      "answer": "Правильный ответ"
    },
    {
      "text": "Вопрос с картинкой",
      "media": {
        "type": "image",
        "src": "/media/questions/round1/example.jpg"
      },
      "answer": "Правильный ответ"
    },
    {
      "text": "Вопрос с ответом в виде картинки",
      "answer": {
        "text": "Правильный ответ",
        "media": {
          "type": "image",
          "src": "/media/answers/round1/answer.jpg"
        }
      }
    }
  ]
}
```

### В супер игре

```json
{
  "name": "Тема супер игры 1",
  "question": "Вопрос супер игры",
  "answer": "Правильный ответ"
},
{
  "name": "Тема супер игры 2",
  "question": "Вопрос супер игры с видео",
  "media": {
    "type": "video",
    "src": "/media/questions/supergame/video.mp4"
  },
  "answer": {
    "text": "Правильный ответ",
    "media": {
      "type": "image",
      "src": "/media/answers/supergame/answer.jpg"
    }
  }
}
```

## Рекомендации по именованию файлов

- Используйте понятные имена: `question1-fireworks.jpg`, `answer-santa.png`
- Избегайте кириллицы и пробелов в именах файлов
- Используйте дефисы `-` вместо пробелов
- Примеры: `new-year-tree.jpg`, `santa-claus-video.mp4`

## Тестирование

После добавления медиа-файлов:
1. Запустите приложение: `npm run dev`
2. Проверьте, что изображения/видео отображаются корректно
3. Убедитесь, что кнопка "Показать ответ" работает

---

**Совет**: Начните с нескольких простых изображений, чтобы убедиться, что всё работает, прежде чем добавлять все медиа-файлы.
